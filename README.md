# Hospital Management SQL Project

## Project Overview
This project is developed using MySQL to manage hospital operations such as patient records, doctor details, appointments, treatments, and billing information.

## Database Tables
- Patients
- Doctors
- Appointments
- Treatments
- Billing

## SQL Concepts Used
- Joins
- Group By
- Aggregate Functions
- Subqueries
- Foreign Keys

## Analysis Performed
- Revenue Analysis
- Payment Analysis
- Patient Insights
- Treatment Analysis
- Billing Analysis

## Tools Used
- MySQL Workbench
- SQL

## Project Highlights
- Designed a relational database using 5 tables
- Managed 150+ records
- Performed hospital data analysis using SQL
